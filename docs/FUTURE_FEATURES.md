# Future Features

This list tracks potential enhancements that are not implemented yet.

## Platform

- macOS backend
- Wayland backend for Linux (in addition to X11)

## Widgets

- DatePicker / TimePicker
- SearchBar with history and suggestions
- TagInput with removable chips
- Stepper / Wizard
- Accordion
- Toolbar
- Notification / Toast system
- RangeSlider (two-handle)
- RichTextPreview (Markdown/HTML)
- Chart widget (line, bar, pie)
- DataGrid with sorting/filtering/pagination
- PropertyGrid (key/value editor)
- VirtualList for very large data sets

## Text and Editing

- Built-in syntax highlighting presets for TextEditor (C++, Python, JSON, etc.)
- Global undo/redo framework (beyond TextEditor)

## Theming and Styling

- Theme import/export (JSON serialization)
- Icon system (SVG/icon font integration)
- Animation framework for widget transitions
- Accessibility options (high contrast, larger fonts, reduced motion)

## Graphics and Effects

- Blur/frosted glass effects
- SVG rendering
- Image cropper
- Canvas widget for custom drawing

## Developer Experience

- Debug overlay for widget bounds and IDs
- Global hotkey manager
- UI test helpers and input simulation
